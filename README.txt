# Climate_Resources

Climate_resources attempts to uncover patterns in resources availability among developed nations due to climate change. A more detailed explanation can be found in Project_Proposal.

## Dependencies

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install wbdata and wbpy.

```bash
pip install wbdata
pip install wbpy
```

## Viewing

Data gathering and orginization can be viewed in Data.ipynb with Jupyter notebook.
The final presentation and analysis can be viewed in Presentation_Keynote or Presentation_Powerpoint.
